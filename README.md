# Learning Caching Headers

Learning `cache-control` headers with serverless, AWS Lambda and AWS Cloudfront.

## Usage

```
yarn
yarn deploy
```

> First deploy can take very long (around 5 minutes) as we need to wait for CloudFront to instanciate
